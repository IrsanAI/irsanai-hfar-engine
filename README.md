### Was ist HFAR?

HFAR ist die **globale, token-sparende Lese- und Optimierungs-Engine** des IrsanAI Universums.  
Sie analysiert jede README (oder beliebigen Text) extrem effizient und schlägt automatisch das einheitliche IrsanAI-Layout vor.

**USP**:  
- Liest nur 25–35 % des Textes  
- Erkennt Struktur, Claim und Lücken  
- Generiert sofort ein optimiertes, standardkonformes README.md  
- Funktioniert bei Repos, Artikeln, Logs, Transkripten, Papers…

---

### So funktioniert die GitHub Page (Live-Pipeline)

1. **Analyse** → HFAR scannt die fremde README  
2. **Layout-Vergleich** → Zeigt fehlende IrsanAI-Elemente  
3. **Optimierung** → Generiert neues, schönes README.md  

**Features der Page**:
- Direkte Eingabe einer beliebigen GitHub-Repo-URL
- Schritt-für-Schritt-Pipeline-Anzeige
- Copy-Button (Profi-Modus)
- Download als `README-optimized.md`
- Mini-Erklärung für Einsteiger
